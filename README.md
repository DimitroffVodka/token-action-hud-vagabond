## Features
Make rolls directly from the HUD instead of opening your character sheet.
Use items from the HUD or right-click an item to open its sheet.
Move the HUD and choose to expand the menus up or down.
Unlock the HUD to customise groups and actions.
Add your own macros and Journal Entry and Roll Table compendiums.

## Installation

On Foundry VTT's Configuration and Setup screen, go to Add-on Modules
Click Install Module
In the Manifest URL field, paste: https://raw.githubusercontent.com/DimitroffVodka/token-action-hud-vagabond/master/module.json
Click Install next to the pasted Manifest URL
Required Modules
Token Action HUD Core

socketlib
Token Action HUD Core requires the socketlib library module.

Recommended Modules
Token Action HUD uses the Color Picker library module for its color picker settings.
